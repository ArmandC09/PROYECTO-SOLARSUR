# SolarSur - Aplicación inicial

Proyecto inicial scaffold para gestionar ventas, cotizaciones, inventario y clientes.

Instalación rápida:

```bash
npm install
npm run dev
```

Estructura principal:
- `src/components/` — componentes principales (Perfil, Proveedores, Cotizaciones, Venta, Inventario, Clientes, Historial).
- `src/App.jsx` — SPA que muestra secciones en una sola página.

Siguientes pasos sugeridos:
- Definir modelo de datos para cotizaciones, ventas y productos.
- Añadir persistencia en backend (API) o mejorar `localStorage`.
- Integrar logo real y paleta de colores definitiva.
